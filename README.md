# Word2Vec (Skip-gram with Negative Sampling) — NumPy Implementation

This repository contains an implementation of Word2Vec using NumPy.

The implemented variant is: Skip-gram model with negative sampling


## Project Structure
```
Word2vec
├── data_sets.py     # data preprocessing 
├── w2v_model.py     # Word2Vec model (embeddings + training step)  
├── training.py      # training loop  
├── tests.py         # training model and test of similarity of words  
├── data/            # text (enwik8)  
└── README.md
```
