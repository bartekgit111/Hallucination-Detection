# Word2Vec (Skip-gram with Negative Sampling) — NumPy Implementation

This repository contains an implementation of Word2Vec using NumPy.

The implemented variant is: Skip-gram model with negative sampling


## Project Structure

.
├── data.py          # data preprocessing
├── model.py         # Word2Vec model (embeddings + training step)  
├── train.py         # training loop  
├── data/            # text (enwik8)  
└── README.md
